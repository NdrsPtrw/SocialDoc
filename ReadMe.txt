SocialDoc ist eine Android-App, die es ermöglicht, die Anzahl und Dauer von sozialen Kontakten zu dokumentieren.

Dazu wird zu Zeiten, die vom User festgelegt werden, ein Alarm gegeben und die App in den Vordergrund geholt. 
Klickt der User dann auf den "?"-Button, gelangt er in den Fragebildschirm, in dem er Angaben über die Anzahl und Dauer seiner sozialen Kontakte seit seiner letzten Antwort machen soll.

Ein neuer User wird angelegt, indem die Datei Probandencode.csv vom Speicher des Smartphones gelöscht wird. In der Regel sollte das durch die Studiendurchführenden gemacht werden, nachdem sie die Probandencode.csv des letzten Users vom Smartphone auf ihren Computer übertragen haben.
Beim nächsten Start der App wird dann eine neue Probandencode.csv angelegt und der neue Nutzer muss seinen Probandencode eingeben und die Alarmzeiten neu wählen.
